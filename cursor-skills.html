ここから先で、スキルの仕組みをひとつずつ丁寧に解説していきます。

スキルって何？

新しいアルバイトが入ってきたとき、あなたはどうしますか？ 毎回「こうやって接客して」「レジはこう打って」と説明するのは大変です。だから多くのお店では「マニュアル（手順書）」を用意しています。アルバイトはマニュアルを読んで、一定の品質で仕事ができるようになります。

CursorのAI（エージェント）も同じです。スキルとは、AIへの「仕事のマニュアル」です。

スキルの役割をアルバイトのマニュアルで考える

お店のマニュアル
アルバイトへの説明書
  • 接客のやり方が書いてある
  • レジの操作手順が書いてある
  • 一度作れば何人でも共有できる
  • 誰が担当しても同じ品質になる
CursorのSKILL.md
AIへの説明書
  • 図解の作り方が書いてある
  • デプロイの手順が書いてある
  • 一度作れば何人でも共有できる
  • 誰が使っても同じ品質になる

ここがポイント

スキルは 「SKILL.md」（スキル・エムディー）というファイルで作ります。このファイルに「こういう作業をするときは、こういう手順でやって」と書いておくだけ。あとはCursorが自動的に読み込んで、AIに覚えさせてくれます。 出典: Cursor 公式ドキュメント — エージェントスキル

スキルはどこに置く？

スキルは、決まった場所にファイルを置くだけでCursorが自動的に見つけてくれます。「どこに置くか」で、使える範囲が変わります。

このプロジェクト（フォルダ）を開いているときだけ使えるスキル。チームでフォルダを共有すれば、全員が同じスキルを使える。

どのプロジェクトを開いていても常に使えるスキル。自分がよく使う作業はここに入れておくと便利。

今あなたが使っているスキルはどこにある？

図解生成スキルは .claude/skills/creating-visual-explainers/SKILL.md に入っています。プロジェクト専用スキルとして、このフォルダを開いているときだけ使える設定になっています。

どうやって使う？

スキルを使うのに特別な操作は必要ありません。Cursorがスキルを自動で読み込んでくれるので、普通に話しかけるだけで動きます。2つの呼び出し方があります。

1
普通に話しかける（自動）

AIが会話の内容から「これはこのスキルを使う場面だな」と自動で判断して動いてくれます。

APIの仕組みを図解して
図解スキルを使って作成します！
2
スキル名で呼び出す（手動）

チャット欄で /（スラッシュ） を入力するとスキル一覧が表示されます。選んで実行できます。

/ creating-visual-explainers
creating-visual-explainers 図解を生成・公開する

今使えるスキルを確認する方法

1
Cursor Settingsを開く WindowsならCtrl+Shift+J
2
「Rules」をクリック 左側のメニューから選ぶ
3
「Agent Decides」を確認 登録済みスキルが一覧で出る

他の人が作ったスキルをインストールする

スキルは自分で作るだけでなく、他の人が公開したスキルをインターネット（GitHub）から取り込んで使うこともできます。アプリのインストールに似たイメージです。

GitHub からスキルをインストールする手順

1
Cursor Settings を開く
キーボードで Ctrl + Shift + J を押す
2
「Rules」→「Add Rule」をクリック
左メニューの「Rules」を開き、「Add Rule」ボタンを押す
3
「Remote Rule (Github)」を選ぶ
表示された選択肢から「Remote Rule」を選択する
4
GitHubのURLを貼り付けて完了
インストールしたいスキルのGitHubリポジトリURLを入力する

スキルはチームで共有できる

プロジェクトフォルダにスキルを入れておけば、そのフォルダをGitで共有するだけでチーム全員が同じスキルを使えます。「このチームでのAIの使い方」をファイルで管理できるのが、スキルの大きなメリットです。

まとめ — 3つのポイント

01

スキル = AIへの仕事の説明書

SKILL.mdというファイルに「この作業はこうやって」と書いておくだけで、AIがいつでもその通りに動いてくれます。毎回長い指示を出す必要がなくなります。

02

決まった場所に置けば自動で動く

.cursor/skills/ フォルダにスキルを置くだけで、Cursorが自動的に読み込みます。特別な設定は不要です。

03

他の人のスキルを使うこともできる

GitHubで公開されているスキルをインストールして使えます。自分で作らなくても、誰かが作ったスキルをそのまま活用できます。

あなたはすでにスキルを使っています

「図解して」と話しかけるだけで図解が完成するのは、creating-visual-explainersスキルが裏側で動いているからです。スキルのおかげで、AIが毎回同じ品質の図解を作れるようになっています。
